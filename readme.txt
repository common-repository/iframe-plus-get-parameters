Passes GET parameters to iframes.

== Description ==
This plugin extends iframes in WordPress and passes GET parameters to the iframe. Can be used to prepopulate forms that pull data from php $_GET variables, also know as the query string. 

== Installation ==
1. Upload this plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
